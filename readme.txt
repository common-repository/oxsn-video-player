This plugin adds a video player shortcode!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/oxsn-video-player` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->OXSN Video Player screen to configure the plugin
